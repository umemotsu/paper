Let $i$ be a non-negative integer. Fibonacchi sequence $F{i}$ has the following property:
\begin{eqnarray*}
F{0} &=& 0, \
F{1} &=& 1, \
F{i+2} &=& F{i} + F{i+1}
\end{eqnarray*}
